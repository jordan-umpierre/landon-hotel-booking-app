# Landon Hotel Booking App

![Landon Hotel Booking App banner](src/main/UI/src/assets/images/hotel/splash_hotelphoto.jpg)

A full-stack hotel reservation demo built with Spring Boot and Angular. Users can search for available rooms by date, create reservations, view localized welcome messages in English and French, and see a shared event time across multiple time zones.

This project started from a starter application and was extended into a portfolio-ready demo focused on backend integration, concurrent localization, and containerized delivery.

## Highlights

- Search room availability by check-in and check-out date
- Create reservations through a Spring REST API
- Load localized welcome messages concurrently with Java threads
- Display the same event time in Eastern, Mountain, and UTC time zones
- Package the application with Docker for straightforward deployment

## Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database
- Angular
- Maven
- Docker

## Run With Docker

```bash
docker build -t landon-hotel-booking-app .
docker run -p 8080:8080 landon-hotel-booking-app
```

Then open `http://localhost:8080`.

## Run Locally

1. Install Java 17 and Node.js 18+.
2. Install frontend dependencies:

```bash
cd src/main/UI
npm ci
```

3. Return to the project root and start the application:

```bash
./mvnw spring-boot:run
```

On Windows PowerShell, use:

```powershell
.\mvnw.cmd spring-boot:run
```

## Test And Verify

Run the backend tests from the project root:

```powershell
.\mvnw.cmd -q "-Dexec.skip=true" test
```

Run the frontend build and headless tests:

```powershell
cd src/main/UI
npm ci
npm run build
npm run test:ci
```

Run a full package build from the project root after installing frontend dependencies:

```powershell
.\mvnw.cmd package
```

The Angular build writes browser assets into `src/main/resources/static`, which is ignored because it is regenerated by the build.

Manual smoke test checklist:

- Open `http://localhost:8080` and confirm the hotel page loads.
- Confirm English and French welcome messages render.
- Confirm ET, MT, and UTC event times render.
- Search for rooms with check-in and check-out dates.
- Reserve a room and confirm the request succeeds.
- Open `http://localhost:8080/h2-console` with JDBC URL `jdbc:h2:file:~/landon-hotel-booking-db`, user `sa`, and a blank password to inspect data.

## API Overview

- `GET /api/messages`
  Returns English and French welcome messages plus the event time in ET, MT, and UTC.
- `GET /room/reservation/v1?checkin=YYYY-MM-DD&checkout=YYYY-MM-DD`
  Returns available rooms for the requested dates.
- `POST /room/reservation/v1`
  Creates a reservation for a selected room.
